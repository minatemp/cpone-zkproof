- [Yarn](https://classic.yarnpkg.com/en/docs/install)
- [Node.js](https://nodejs.org/en/)
- [Auro extension wallet](https://www.aurowallet.com/)

## Running the module

You can use the following commands to run the module:

```bash
yarn install
yarn run dev
```
